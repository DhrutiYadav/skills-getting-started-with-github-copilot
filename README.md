## Run locally

Use the helper scripts to start the app quickly:

```bash
make run
```

or:

```bash
./run.sh
```

Both commands will create a `.venv` virtual environment, install dependencies from `requirements.txt`, and start the FastAPI app with `uvicorn`.

Then open:

```text
http://127.0.0.1:8000/
```
